# Changelog

## [1.0.0] - 2024-01-XX

### Added
- Initial release of AWS CLI Assistant Lite
- Natural language to AWS CLI command conversion
- Support for S3, EC2, DynamoDB, Lambda, and IAM services
- MCP (Model Context Protocol) server implementation
- HTTP adapter for REST API access
- Real-time command validation
- Configurable ML and rule-based NLP processing

### Features
- Rule-based intent parsing with ML fallback
- AWS CLI command generation for common operations
- Input validation and error handling
- Telemetry and logging support
- Cross-platform compatibility (Windows, Linux, macOS)
